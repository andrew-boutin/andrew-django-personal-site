Board
The board is a tenkeyless 87 key style ordered off of Winkeyless. It comes soldered together with the main controller parts, RGB LED under board lighting, RGB controller, dioes, and resistors.

Switches
The switches are Gateron Browns 45g tactile from Massdrop. Browns are softer (less clicky) and have a feedback bump around halfway through the key press.

Key LEDs
The key LEDs from switchTop are blue.

Keycaps
The Dolch 110 key base set is from Pimp My Keyboard and includes extra keycaps for customization. The core keys are gray, and the secondary keys are black. There are 110 keys included in the set which allows some of the secondary keys to be either black or grey. There's also a red escape key included in the set.

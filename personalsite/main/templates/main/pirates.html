Pirates Dev Info
  • Technology:
    This game was developed using the Unity Engine. All coding/scripting was in C# using MonoDevelop for an IDE, and uLink was used for all of the networking.
    Dev. Team Structure:
    When I was working on this project there were four coders including myself and four 3D artists. One of the artists was also the lead developer who took care of all social media, recruiting, and game concepts. A lot of game mechanics were decided as a group when the issues came up. Skype was utilized for all communication and an SVN repository was hosted for the project builds.
    My Role:
    I was a member from January to September 2014. I was brought on to the team after applying to be a coder and creating a demo project. Here are some of my main contributions:
    • Mecanim layer to handle combat animation transitions
    • Weapon "class" system - weapon | rangedWeapon, meleeWeapon | basic weapon scripts
    • Generic weapon, melee, and ranged management scripts
    • Raycasting, hit detection, sound, and particle systems for ranged weapons
    • Skeleton mask to control/combine combat vs movement animations
    • Assisted with networking to display combat animations across all clients
    • Assisted with architecture design for the multiple game servers

Squares Clone Dev Info
  • Technology:
    This game was developed in Python utilizing Pygame. The whole point of creating this game was to try out using Pygame for game development.
    Info:
    The game play and mechanics are very simple. On top of that Pygame provides lots of built in functionality to make game development easier. It handles almost all of the collision detection and graphics rendering. I highly recommend Pygame to developers who want to be able to rapidly develop games without spending time on the UI.
